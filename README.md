# Beer Scholar — Homepage Mockups (Preview)

Three homepage design directions for the 2026 Beer Scholar site rebuild. Static HTML — open
`index.html` to compare, then click into each direction. Built with the real brand logo files,
the official brand color palette, and real copy from the current site.

## Directions
- **A — The Authority** (`direction-a-authority.html`) — navy + gold, scholarly, credibility-first.
- **B — The Closer** (`direction-b-closer.html`) — urgency + pricing cards, conversion-optimized.
- **C — The Craft Guide** (`direction-c-craft.html`) — green/gold, warm editorial, story-led.

## Notes for reviewers
These are **early directions to react to**, not finished pages. CTAs are placeholders and the
opt-in forms don't submit anywhere yet. The goal is to pick a look and feel (mixing elements
across directions is welcome) before the full site is built.

## Brand
Colors: gold `#AE841F` · navy `#02394F` · green `#224C37` · burnt orange `#AF561B` · yellow `#F3C645`.
Web font: Raleway (Direction C also uses Fraunces for headlines).
